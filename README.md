# WORDPRESS REST API Endpoints Contact
Contact endpoints for the WP REST API

**Get All Forms:**
```
GET /wp-json/wp-api-contact/v1/forms
```

**Get Single Form:**
```
GET /wp-json/wp-api-contact/v1/forms/(?P<id>[\d]+)
```

**Create Form:**
```
POST /wp-json/wp-api-contact/v1/forms
```

**Update Form:**
```
PUT /wp-json/wp-api-contact/v1/forms/(?P<id>[\d]+)
PATCH /wp-json/wp-api-contact/v1/forms/(?P<id>[\d]+)
```

**Delete Form:**
```php
DELETE /wp-json/wp-api-contact/v1/forms
```
